# 项目说明

最新的自动爬取微信公众号的实时更新文章、历史文章，视频号的列表、详情、下载的爬虫，帮助广大站长、懒虫、社会精英、商界达人提高工作效率。

支持Linux、Windows、Mac系统，可无限扩展爬取数量、提高爬取速度，无被封风险。

## 主要功能

1.  爬取公众号的实时更新文章（准确性和实时性远超搜狗微信搜索功能） 
2.  根据关键词（标题、正文、作者），爬取某些文章
3.  阅读量，点赞量监控，以及评论采集
4.  爬取历史文章
5.  视频号列表、指标、下载

2021年至今（2023年7月）
